# kalkulator-hpp
Aplikasi web untuk menghitung Harga Pokok Produksi dengan fokus pada bahan baku saja Dilengkapi fitur login multi-user, manajemen produk dan bahan baku (CRUD), kalkulator HPP otomatis,  auto-save ke localStorage, serta UI modern yang responsif. Cocok untuk pemilik UMKM dan  produsen yang ingin menghitung biaya produksi tanpa perhitungan packaging.
